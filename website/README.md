# Sitio web ALIVIO 360

Este sitio estatico contiene la informacion publica del negocio para usarla como
pagina oficial ante Meta:

- Nombre comercial
- Servicios
- Tarifas
- Direccion
- Telefono
- Instagram y enlace de WhatsApp

Para que sirva en la verificacion de Meta, debe publicarse en una URL valida del
negocio, idealmente con dominio propio como `alivio360.co` o similar. Una vez
publicado, esa URL se puede pegar en el formulario de verificacion empresarial.
