## Preguntas para recopilar con el cliente

### Servicios

- ¿Qué servicios ofrecen?
- ¿Qué incluye y qué no incluye cada servicio?
- ¿Cuánto cuesta cada servicio y el precio puede variar?
- ¿Cuánto dura o cuál es el tiempo estimado?
- ¿Se necesita reserva, valoración o requisito previo?

### Horarios y ubicaciones

- ¿Cuál es el horario habitual y el de días festivos?
- ¿Dónde están ubicados y cómo se llega?
- ¿Cada sede ofrece los mismos servicios?
- ¿Atienden a domicilio o de forma virtual?

### Reservas y disponibilidad

- ¿Cómo se agenda, cambia o cancela una cita?
- ¿Con cuánto tiempo se debe cancelar?
- ¿El bot puede confirmar disponibilidad o debe intervenir un asesor?
- ¿Qué información se necesita para reservar?

### Pagos y promociones

- ¿Qué métodos de pago aceptan?
- ¿Se requiere anticipo?
- ¿Cómo funcionan los reembolsos?
- ¿Qué promociones están vigentes y cuáles son sus condiciones?

### Atención y políticas

- ¿Cuándo debe intervenir un asesor?
- ¿Cuál es el tiempo esperado de respuesta humana?
- ¿Cómo se atienden quejas o reclamos?
- ¿Qué solicitudes no puede gestionar el chatbot?

- [ ] Se probaron distintas formas de hacer la misma pregunta.
